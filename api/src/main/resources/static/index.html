Opinionated

Ideell is an opinionated organization with the following principes

  • Pro Democracy
  • Pro Science
  • Pacifist

Moderation

Ideell is moderating as much content as possible by humans. We don't beleive in automated moderation or automated recommendations.

The purpose of the content is the main most important aspect.

Publication

Ideell thinks all content should follow Pressens publicitetsregler (Swe. the press's publication rules). Which are in general

  • News should be correct
  • Respond to etnic and fact criticism
  • Respect privacy
  • Images should not mislead
  • Be objective and include several opinions
  • Be careful with names
